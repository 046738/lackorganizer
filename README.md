# LACKorganizer

FreeCAD-based project for organizing and designing LACK-based solutions.

## Project Structure

```
lackorganizer/
├── CAD/
│   ├── LACKDock.FCStd
│   ├── Base/
│   ├── Modules/
│   ├── DeviceModels/
│   ├── Tests/
│   └── Drawings/
├── STL/
├── STEP/
├── Docs/
├── Images/
├── BOM/
└── README.md
```

## Directories

- **CAD/**: FreeCAD design files and models
- **STL/**: STL export files for 3D printing
- **STEP/**: STEP export files for CAD interchange
- **Docs/**: Documentation and guides
- **Images/**: Project images and renders
- **BOM/**: Bill of Materials
