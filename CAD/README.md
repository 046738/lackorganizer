# CAD Design Files

This directory contains FreeCAD design files for the LACKDock project.

## Files

- **LACKDock.FCStd**: Main FreeCAD design file

## Subdirectories

- **Base/**: Base components and assemblies
- **Modules/**: Modular components that can be combined
- **DeviceModels/**: Device-specific model variations
- **Tests/**: Experimental designs and test iterations
- **Drawings/**: Technical drawings and exported views

## Getting Started

1. Download and install [FreeCAD](https://www.freecadweb.org/)
2. Open `LACKDock.FCStd` in FreeCAD
3. Explore the design hierarchy in the model tree

## Exporting Files

- Export to STL format for 3D printing: File → Export → `.stl`
- Export to STEP format for CAD interchange: File → Export → `.step`
